# Описание основных сущностей
 - TEngine (src/engine) - движок для вывода таблицы в .csv/.jf
 - Работа с csv файлами - src/csvio
 - Колонки (src/columns) в папке /types лежат реализации колонок, в /operators - действия, которые можно делать с колонками с помощью диспатчера Do
 - TExecutor (src/executor) - там лежит парсер SQL и создание плана запуска запроса и пайплайна выполнения. Непосредственно TExecutor этот пайплайн и запускает
 - IosFactory (src/ios_factory) - глобальный синглтон с всеми инпутами и аутпутами. В целом сейчас он не нужен, но особо не мешает и в целом удобен, так что пусть  будет
 - src/memory - не используется
 - utils/ - тайпдефы, вектора. Там еще есть запромпченная хэшмапа и хэшсет, но они не используются
 - Воркеры (src/workers) - элементы пайплайна. Там лежит интерфейс воркеров и в субдиректориях реализация самих воркеров. Все воркеры кроме AoEngine - элементы пайплайна запроса. (сори за нейминг, я не придумал как лучше сделать (Agregation/Operator)Engine). AoEngine нужен для агрегатора и группировки, а именно чтобы считать операторы на колонках (+, -, Extract, If итд), или агрегации
